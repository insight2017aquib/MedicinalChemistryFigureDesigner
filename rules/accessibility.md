# Accessibility Rules

## Purpose

Define accessibility requirements for figures to ensure readability across audiences, including color-blind users and print reproduction.

## Scope

**In scope:**

- Color contrast requirements (placeholder thresholds)
- Color-blind-safe design constraints
- Text legibility minimums
- Non-color-dependent encoding requirements
- Print and screen readability guidelines

**Out of scope:**

- Alt-text content for specific figures
- Specific WCAG compliance certification
- Color values (see `styles/color-system.md`)
