# Example Index

## Purpose

Maintain a catalog of figure examples added to this repository, with metadata linking each example to its template, styles, and validation path.

## Scope

**In scope:**

- Index table structure (placeholder columns)
- Example categorization by template type
- Status tracking (planned, in-progress, complete)
- Attribution and source requirements for each entry

**Out of scope:**

- Populated example entries with scientific content
- Screenshots or asset files (until supplied by maintainers)
