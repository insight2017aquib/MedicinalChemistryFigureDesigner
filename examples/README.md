# Examples Module

## Purpose

Provide a structured location for worked figure examples, before/after comparisons, and annotated specimens once content is supplied by maintainers.

## Scope

**In scope:**

- Example index and categorization
- Placeholder structure for future specimens
- Guidelines for adding examples without embedding fabricated science
- Cross-references to templates and styles used per example

**Out of scope:**

- Fabricated scientific figure content
- Biology or chemistry-specific demonstrations
- Journal-specific example figures
