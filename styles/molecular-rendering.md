# Molecular Rendering

## Purpose

Establish visual conventions for depicting molecular structures, bonds, atoms, and related graphical elements in figures. Provides a style layer separate from scientific accuracy rules.

## Scope

**In scope:**

- Bond line weights and styles
- Atom label formatting conventions (visual only)
- Highlighting and emphasis styles for structural features
- Color application for structural elements (references `color-system.md`)
- Consistency rules across panels

**Out of scope:**

- Chemical accuracy, stereochemistry, or nomenclature
- Software-specific rendering instructions
- Structure drawing algorithms
