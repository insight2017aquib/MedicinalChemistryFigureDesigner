# Typography

## Purpose

Specify typeface selection, sizing scales, weight usage, and text hierarchy for all figure text elements including labels, legends, and annotations.

## Scope

**In scope:**

- Font family slots (heading, body, annotation, chemical notation)
- Size scale and line-height conventions
- Weight and style usage rules
- Panel label and subfigure lettering conventions (structure only)

**Out of scope:**

- Actual font files or licensing details
- Scientific nomenclature or terminology
- Caption text for specific journals
