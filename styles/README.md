# Styles Module

## Purpose

Centralize visual design specifications for figures produced by this skill. Styles are modular and composable; individual style files define discrete aspects of the visual language.

## Scope

**In scope:**

- Color systems, typography, layout grids, molecular rendering conventions, and annotation styles
- Cross-references between style components
- Extension guidelines for new style files

**Out of scope:**

- Scientific labeling content
- Template structure (see `templates/`)
- Validation rules (see `validation/`)
