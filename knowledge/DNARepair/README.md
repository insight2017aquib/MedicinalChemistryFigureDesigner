# DNA Repair Knowledge Pack

## Purpose

Reserve a knowledge pack slot for domain-specific conventions, terminology references, and figure patterns related to DNA repair review articles.

## Scope

**In scope:**

- Pack metadata and versioning structure
- Slots for user-supplied terminology glossaries
- Slots for figure type categorization (structure only)
- Cross-references to `styles/` and `templates/`

**Out of scope:**

- DNA repair pathway descriptions or mechanism content
- Gene names, protein functions, or biological processes
- Journal-specific formatting rules
