# General Design Knowledge Pack

## Purpose

Reserve a knowledge pack slot for cross-domain design conventions, reusable figure patterns, and general publication-quality guidance that applies across scientific domains.

## Scope

**In scope:**

- Pack metadata and versioning structure
- Cross-domain figure pattern index (structure only)
- References to `styles/`, `rules/`, and `templates/`
- General design heuristics placeholders (non-scientific)

**Out of scope:**

- Domain-specific biology or chemistry content
- Journal-specific requirements (see `knowledge/JournalStyles/`)
- Visual style token values (see `styles/`)
