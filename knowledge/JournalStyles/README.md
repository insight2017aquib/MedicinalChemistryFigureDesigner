# Journal Styles Knowledge Pack

## Purpose

Reserve a knowledge pack slot for user-supplied journal and publisher style requirements that govern figure formatting, dimensions, and export conventions.

## Scope

**In scope:**

- Pack metadata and attribution for externally sourced guidelines
- Template structure for documenting user-provided journal requirements
- Reference linking format (URL, document name, date accessed)
- Override mapping to `rules/` and `validation/` modules

**Out of scope:**

- Invented or fabricated journal guidelines
- Default requirements for any specific publisher
- Caption or submission text templates with scientific content
