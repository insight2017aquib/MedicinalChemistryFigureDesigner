# File Naming

## Purpose

Define naming conventions for figure files, variants, and exported assets to ensure consistency and traceability.

## Scope

**In scope:**

- Base filename structure
- Version and revision suffix conventions
- Panel and variant naming patterns
- Character set and length constraints
- Directory organization for exports

**Out of scope:**

- Manuscript figure numbering (user-defined)
- Content-descriptive names tied to science
