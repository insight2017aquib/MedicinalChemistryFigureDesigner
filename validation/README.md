# Validation Module

## Purpose

Define quality gates and checklists applied to figures before export. Validation enforces rules and style compliance without assessing scientific accuracy.

## Scope

**In scope:**

- Pre-export checklists
- Resolution and DPI requirements
- File naming conventions
- Metadata requirements
- Validation severity and pass/fail criteria

**Out of scope:**

- Scientific content verification
- Peer review or journal acceptance criteria
- Automated testing infrastructure (future extension)
