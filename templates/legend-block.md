# Legend Block Template

## Purpose

Define a reusable legend block structure that can be embedded in or appended to any figure template.

## Scope

**In scope:**

- Legend zone layout and sizing
- Entry row structure (symbol + label slot)
- Ordering and grouping conventions
- Placement options relative to figure panels
- Style and rule references for this template

**Out of scope:**

- Legend entry content or scientific symbols
- Caption text
