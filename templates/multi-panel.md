# Multi-Panel Template

## Purpose

Define the layout structure for figures with two or more panels arranged in a grid or custom configuration.

## Scope

**In scope:**

- Panel grid configurations (2-up, 3-up, 2×2, etc.—structure only)
- Panel label zone assignments
- Inter-panel spacing and alignment
- Shared legend placement options
- Style and rule references for this template

**Out of scope:**

- Panel content
- Scientific comparison logic
