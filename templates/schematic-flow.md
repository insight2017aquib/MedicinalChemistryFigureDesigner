# Schematic Flow Template

## Purpose

Define the layout structure for figures depicting sequential or branching process flows using nodes, connectors, and directional indicators.

## Scope

**In scope:**

- Node zone definitions
- Connector and arrow pathway slots
- Flow direction conventions (structure only)
- Branching and merge point placeholders
- Style and rule references for this template

**Out of scope:**

- Process steps or biological/chemical pathways
- Node labels or step descriptions
