# Templates Module

## Purpose

Provide reusable figure layout templates that combine styles and rules into structured starting points for common figure architectures.

## Scope

**In scope:**

- Template index and selection guidance
- Layout structure definitions (slots, zones, panels)
- Template-to-style and template-to-rule mappings
- Extension guidelines for new templates

**Out of scope:**

- Scientific diagram content
- Finished figure examples (see `examples/`)
- Prompt text (see `prompts/`)
