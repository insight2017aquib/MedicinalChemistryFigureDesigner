# MedicinalChemistryFigureDesigner

## Purpose

Provide a modular, GitHub-ready scaffold for a Claude Skill that designs publication-quality scientific figures for medicinal chemistry and molecular biology review articles. This repository defines architecture, conventions, and extension points—not scientific content.

## Scope

**In scope:**

- Folder layout and module boundaries
- Placeholder documentation for styles, rules, templates, validation, prompts, and examples
- Entry points for Claude (`CLAUDE.md`, `instructions.md`)
- Extension guidelines for maintainers

**Out of scope:**

- Scientific content, biology, or chemistry specifics
- Journal-specific submission guidelines
- Finished figure assets or worked examples
